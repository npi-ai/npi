# Google Calender

## Usage

<Steps>

    ### Authorization

    ```sh
    npi auth google calendar --secret-file <path-to-secret-file>
    ```

    ### Usage

    <Tabs items={["CLI", "Python"]}>
        <Tabs.Tab title="CLI">
            ```sh
            npi app gcal "is example@example.com free on next monday at 3pm?"
            ```
        </Tabs.Tab>
        <Tabs.Tab title="Python">
            ```python
            from npiai.app.google import Calendar
            calendar = Calendar()
            calendar.chat("is example@example.com free on next monday at 3pm?")
            ```
        </Tabs.Tab>
    </Tabs>

</Steps>

## Examples

### Check availability

```python
calendar.chat("is example@example.com free on next monday at 3pm?")
```

### Create an event
```python
calendar.chat("create an event to invite a@a.com, b@b.com to discuss about project on next monday at 3pm?")
```
