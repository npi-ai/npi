# Twitter

The Twitter App is designed to streamline the process of accessing and managing Twitter data. Users can interact with tweets in a variety of ways directly through the app's functions.

## Setup

<Steps>

    ### Authorization

    <Callout type='info'>
      The Twitter Integration mimics browser-based login flow, requiring your Twitter credentials for secure interaction. Please initialize the app by providing your username and password.
    </Callout>

    ```
    npi auth twitter --username <twitter-username> --password <twitter-password>
    ```

    ### Usage

    <Tabs items={["CLI", "Python"]}>
        <Tabs.Tab title="CLI">
            ```sh
            npi app twitter "Find and summarize the latest tweet by @AtomSilverman discussing AI agents."
            ```
        </Tabs.Tab>
        <Tabs.Tab title="Python">
            ```python
            from npiai.browser_app.twitter import Twitter
            twitter = Twitter()
            twitter.chat('Find and summarize the latest tweet by @AtomSilverman discussing AI agents.')
            ```
        </Tabs.Tab>
    </Tabs>

</Steps>
