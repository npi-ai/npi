# General Browser Agent

The General Browser Agent is a flexible, browser-based automation tool capable of performing a wide range of tasks on various webpages.

<Callout type='info'>
    The General Browser Agent is designed for general navigation purposes. You may want a
    fine-grained optimization when targeting a specific website.
</Callout>

## Usage

    <Tabs items={["CLI", "Python"]}>
        <Tabs.Tab title="CLI">
            ```sh
            npi app browser "Order a Big Mac from McDonald's near 10011 using Doordash."
            ```
        </Tabs.Tab>
        <Tabs.Tab title="Python">
            ```python
            from npiai.browser_app.browser import Browser
            browser = Browser()
            browser.chat("Order a Big Mac from McDonald's near 10011 using Doordash.")
            ```
        </Tabs.Tab>
    </Tabs>
